Atividade proposta pela faculdade de BSI 2º termo.

--------------------------------------------------------------------------------------------------
Basicamente é um sistema de uma pizzaria com controle de motoqueiro, pizzas, cliente e pedidos
tudo salvo em arquivos.

Conta também com relatórios, exclusões físicas e lógicas, buscas variadas, como direta, indireta,
ordenação de bolha, inserção direta entre outras..

Para navegar entre os campos é auto explicativo pois tem uma interface condizente.

--------------------------------------------------------------------------------------------------

Ao escolher algo do menu de opções [1], [2], [3], para sair é só apertar ENTER. Para sair do
programa tem que estar dentro do menu de opções e apertar ESC.

Após ter entrado em alguma opção do menu de opções por exemplo, [1], irá aparecer outro menu no centro
da tela, ao clicar em [Q] ou [W], para sair basta clicar ENTER.

Já se clicou no [E] ou [R] para sair tem que apertar 0 e dar enter, isso vale para todos do menu de opções.

--------------------------------------------------------------------------------------------------

Algumas opções tem que escolher (S/N), é nada mais que SIM ou NÃO, basta clicar S ou N.

--------------------------------------------------------------------------------------------------

No menu de opções ao escolher o [7], aparece escolhas adicionais.

[Q] escolher a data inicial xx/xx/xxxx de uma pizza e a final para ver todas as pizzas feitas naquele periodo.
para sair tem que digitar 0 dar enter 0 dar enter 0 dar enter. 

(A partir do primeiro 0, os números sairão para fora da tela, mas ainda está funcionando, considere como bug)

[W] Digite a letra inicial do nome de alguém que queira ver, se estiver cadastrado irá mostrar.

[E] Mostra as pizzas da mais vendida para menos vendida.

[T] Mostra se as pizzas pedidas se estão em preparo, a caminho ou entregue.

[R] Mostra o gasto de todos os cliente que mais gastou comprando pizzas.

[0] Adiciona dados de todos os campos, clientes, motoqueiros, pizzas de forma automatica, para testes.
